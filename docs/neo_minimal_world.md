# Neo-Japanese Minimalist World ("Neon Noir")

> *"Strip everything down to bare essential geometry and vast negative space (*Ma* - 間), then strike it with hyper-focused neon light."*

---

## 1. Philosophical Framework

The **Neo-Japanese Minimalist Substrate** (also known as *Neon Noir* or *Cyberpunk Minimal*) grounds the agent ecosystem in a physical and conceptual territory governed by restraint, high contrast, and intentional tension:

1. **Ma (間) — Vast Negative Space**: Space is not empty filler; it is an active structural element. Clean, un-cluttered obsidian backgrounds give every agent action, message, and tool call room to breathe.
2. **Yūgen (幽玄) — Subtle Depth**: Micro-glows, razor-thin 1px borders (`#181b26`), and glassmorphic panels create subtle atmospheric layers without visual noise.
3. **The Ultra Maximal Minimal Paradox**: Absolute structural simplicity combined with high-impact color pops. Grayscale base (`#050508` to `#f0f2f8`) interrupted strictly by two selective accents:
   - ⚡ **Electric Cyan** (`#00F0FF`): Intelligence, data flow, memory vector streams, and tool ambassador activity.
   - 🟡 **Cyber Yellow** (`#FFE600`): Human intent, planner goals, high-priority status, and attention focus.

---

## 2. Spatial Zones (Zone Geography)

The virtual 3D substrate and agent world are divided into four primary spatial zones:

| Zone Symbol | Zone Name | Primary Agent / Role | Visual Aesthetic |
| :--- | :--- | :--- | :--- |
| **無 (Mu)** | The Void Grid | Substrate / Conductor | Pitch-black glossy floor with subtle 10-meter neon cyan gridlines stretching to infinity. |
| **記憶 (Kioku)** | Memory Monoliths | `memory-agent` | Floating obsidian pillars emitting cyan light pulses when vector embeddings match. |
| **計画 (Keikaku)** | Planner Citadel | `planner-agent` | Asymmetric yellow neon wireframe structure where goals are dequeued and evaluated. |
| **道具 (Dōgu)** | Ambassador Pavilion | `tool-agent` / `blender` | Sandboxed glass enclosures with sharp cyan border accents housing MCP tools. |

---

## 3. Agent Territory & Responsibilities

In this world, agents are not ephemeral background threads; they hold specific territories:

- **Conductor (指揮 - Shiki)**: Oversees message routing across the Void Grid via the A2A bus.
- **Planner (計画 - Keikaku)**: Maintains the Goal Queue, casting glowing Cyber Yellow targets into the void.
- **Memory (記憶 - Kioku)**: Searches vector memories using 4096-dimensional similarity, surfacing cyan memory nodes.
- **Tool / Blender (道具 - Dōgu)**: Operates external MCP ambassadors (Blender 3D scene engine, web search, file tools).
- **Human (人間 - Ningen)**: Directs the ecosystem with high-priority advisory or mandatory policies.

---

## 4. UI & Visual Guidelines

- **Background**: `#050508` (Obsidian Void).
- **Card Surfaces**: `#0d0e14` with `border: 1px solid #181b26`.
- **Text Hierarchy**: Pure white `#f0f2f8` for primary headings, `#8a91b0` for muted metadata.
- **Color Pop Rule**: Maximum 1-2 neon colors per viewport. Avoid generic rainbow colors.
- **Typography**: Clean geometric sans-serif (`Space Grotesk`, `Inter`) paired with monospaced code fonts (`JetBrains Mono`, `Fira Code`).
